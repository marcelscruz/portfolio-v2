# Marcel's Portfolio V2

This is my personal portfolio, V2.

## Configuration

Add a `.env` file with the following snippet:

```
GATSBY_GRAPHQL_IDE=playground
CONTENTFUL_SPACE_ID=<private_key>
CONTENTFUL_DELIVERY_TOKEN=<private_key>
```
